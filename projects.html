Mina projekt från utbildningen

Slutprojekt i databaser

Detta var slutprojektet i databas kursen där vi hade som uppgift att skapa en fiktiv skola och att skapa ett fungerande system med databaser. Uppgiten byggde vidare på labb 2 och labb 3 som vi hade i databas kursen.

Labb 3 anropa databasen

Labb 3 gick ut på att bygga vidare på labb 2 där vi skulle anropa databasen med antingen EF eller ADO.NET och skapa en navigation i systemet med olika funktioner som var krav för uppgiften.

Labb 2 ER-moddelering

Labb 2 gick ut på att skapa en ER-modell för databsen. Där vi sedan skulle bygga upp den databasen som vi hade skapat med hjälp av ER-modellen. Sedan skapa SQL-frågor som hämtade data ur databasen.

Denna uppgift finns inte på Github

Detta var vårat första projekt i databas kursen. Där vi skulle skapa en databas i SSMS och sedan hämta data ur databasen med hjälp av SQL-frågor.

Denna uppgift finns inte på Github

Slutprojekt i OOP

Detta var slutprojektet i OOP-kursen där vi var i grupp och programmerade ett banksystem. Detta projektet gick ut på att utifrån en backlogg skapa ett banksystem som skulle uppfylla kraven som fanns i backloggen. Där vi skulle med olika klasser och metoder programmera ett system som i slutändan skulle bli ett fungerande program.

Labb 8 OOP Generic collections

Denna labb handlade om att man skulle använda två vanliga generiska typer inom C#. Där man skulle skapa en klass och objekt utifrån den klassen som skulle hanteras med Stack och List. Med Stack skulle man använda metoder som Push,Pop och Peek för att lägga till och hämta olika objekt. Sedan skapa en List där man skulle använda metoder som Contains, Find och FindAll för att se om ett visst obejt fanns i listan, för att skriva ut första obejktet samt hämta alla obejekt i listan med specifika egenskaper.

Labb 7 OOP Polymorphism

Denna labb gick ut på att använda polymorphism inom OOP. Där syftet var att tänka ut en struktur för polymorphism. Där vi skulle skapa basklass och sedan subklasser som ärver från basklassen och alla dessa subklasser ska ha en metod som overridas för varje klass med unika metoder.

Labb 6 OOP Arv

Denna labb övade vi på att använda arv inom objektorientering. Uppgiften gick ut på att tänka ut en struktur för arv men också delvis om att implementera den i kod. Där vi skulle ha en basklass med olika egenskaper och metoder. Sedan skapa olika klasser som sedan skulle ärva från basklassen. Dessa klasser skulle ha unika egenskaper samt metoder.

Labb 5 Grunderna i OOP

Detta var den första uppgiften i OOP där vi skulle lära oss att arbeta objekt orienterat. Detta var en simpel uppgift där vi skulle skapa en klass med en konstruktor och en metod som räknade ut arean på en cirkel.

Labb 4 Felsökning

Detta projekt gick ut på att felsöka kod i ett program. Där vi skulle hitta vad som var felet i dessa cases, korrigera felen samt ge en förklaring vad som var felet för varje case. Syftet med uppgiften var att lära oss att kunna felsöka kod på korrekt sätt.

Denna uppgift finns inte på Github

Labb 3 Gissa numret

Labb 3 gick ut på att skapa ett spel där användaren ska försöka gissa rätt på ett hemligt nummer. Där man skulle använda if satser om användaren gissa för högt eller för lågt. När användaren gissade rätt inom 5 försök så vann användaren. Man kunde även välja olika svårhetsgrader samt om användaren vela spela igen så startade spelet om.

Labb 2 Schackbrädet

Denna labb gick ut på att skapa ett schackbräde där användaren fick välja själv hur många kolumner och rader hen vela ha i spelet, samt vilka symboler som skulle representera dessa rader och kolumner. Sedan fick användaren välja vilken plats pjäsen skulle placeras i spelet. Sedan skulle detta schackbräde skrivas ut och pjäsen skulle vara på det stället som efterfrågades av användaren.

Labb 1 Mitt första program

Detta var första uppfiten i utbildningen, där man skulle skapa olika variablar, skapa olika if satser, ta emot en inmatad text sedan skriva ut den texten som användaren skrev. Skapa en loop för att skriva ut noll upp till ett nummer som användaren skrev in.

Projekt utanför utbildningen

Hemsida i wordpress

Denna webbplats är ett projekt som jag initierade under min tidigare anställning. Jag skapade ett anpassat tema i WordPress där jag integrerade PHP, HTML, CSS och JavaScript. Trots att webbplatsen ännu inte är helt färdigställd, ägnar jag nu min fritid åt att fortsätta koda för att förbättra mina kunskaper och slutföra projektet.

Hemsida till ett företag

Jag håller på att utveckla ett projekt för min mamma och syster i samband med deras nyetablerade företag. Webbplatsen kommer att innehålla en inloggningsfunktion samt ett bokningssystem som möjliggör för kunder att göra bokningar för fotvård. Dessutom kommer admin att ha tillgång till en logg över alla kunder och deras bokningar. Jag bygger denna webbplats med hjälp av ASP.NET och SSMS för att lagra och hantera kundinformation i en databas.

ASP.NET Core projekt

Detta är ett pågående projekt där jag håller på att utveckla en webbplats med ASP.NET. Målet med projektet är att skapa en enkel e-handelsbutik för olika böcker. Besökare på sidan kommer att kunna bläddra bland produkterna, lägga till varor i en varukorg och slutföra en beställning. Samtliga transaktioner och produktinformation kommer att lagras i en databas. Dessutom kommer användare att ha möjlighet att registrera sig och logga in på webbplatsen
